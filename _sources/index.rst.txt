=========================
TII Cryptanalysis Library
=========================

This is a sample reference manual for TII cryptanalysis library.

To use this module, you need to import it:: 

    from cryptographic_estimators import *

This reference shows a minimal example of documentation of the
TII cryptanalysis library following SageMath guidelines.
